===== COMMENT LANCER L APPLICATION =================

1) Extraire le projet de l'archive 
2) Creer un nouveau projet Eclipse  et y inclure le dossier de l'archive
3) Mettre la librairie mysql-connector-java dans le projet 
4) Renseigner les champs dans le fichier DBConfig.java  pour accéder à la base de données qui se trouve sur le site http://webtp.fil.univ-lille1.fr/
5) Executez le MainProg.java
6)  Une fois l’application exécutée, deux choix s’offre à vous :
- soit se connecter en mode admin (vous aurez la possibilité de créer, modifier et supprimer un utilisateur  )
- soit se connecter en mode utilisateur ( vous pourrez créer un groupe, ajouter des amis et discuter avec eux .. )


Les identifiants administrateurs sont :
Pseudo : admin
mot de passe : admin
